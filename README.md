## 👨‍💻 About Me

I am a **DevOps & Cloud Engineer** specializing in automated infrastructure provisioning, container orchestration, and continuous integration/continuous delivery (CI/CD) pipelines.

With hands-on experience in **AWS**, **Terraform**, **Docker**, **Kubernetes**, and **Linux**, I focus on engineering resilient multi-tier cloud architectures, eliminating manual overhead through Infrastructure as Code (IaC), and establishing production-grade deployment workflows.

- ☁️ **Cloud Architecture:** Designing and deploying secure, high-availability multi-AZ environments on AWS (VPC, EC2, ALB, RDS, Auto Scaling).
- 🏗️ **Infrastructure as Code:** Automating cloud resource lifecycle management using modular Terraform templates.
- 🐳 **Containerization & Orchestration:** Packaging applications with Docker and managing workload deployments in Kubernetes environments.
- 🔄 **CI/CD & Automation:** Building continuous integration and automated deployment pipelines with Jenkins and GitHub Actions.
