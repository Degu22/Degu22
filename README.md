## 👨‍💻 About Me

I am a passionate **DevOps & Cloud Engineer** focused on designing, automating, and deploying high-availability cloud infrastructure and robust CI/CD pipelines. 

My work centers around **Infrastructure as Code (IaC)** with Terraform, cloud provisioning on **AWS**, container orchestration with **Docker & Kubernetes**, and automated delivery pipelines using **Jenkins** and **GitHub Actions**. I strive to build scalable, fault-tolerant systems with clean code, standard-compliant architecture, and complete operational transparency.

- 🌩️ **Cloud Expertise:** Designing multi-tier, multi-AZ architectures on Amazon Web Services (VPC, EC2, ALB, Auto Scaling, RDS).
- ⚙️ **Infrastructure Automation:** Writing modular, reusable Terraform scripts for seamless environment provisioning.
- 🐳 **Containerization & Orchestration:** Packaging applications with Docker and orchestrating workloads using Kubernetes.
- 🔄 **Continuous Delivery:** Creating automated build, test, and deployment workflows with Jenkins and GitHub Actions.
- 💻 **Full-Stack Awareness:** Building modern developer interfaces using Next.js and TypeScript.
